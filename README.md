Division with Promises
This project demonstrates how to use JavaScript Promises with an arrow function to perform division between two numbers.
The function handles division by zero gracefully by rejecting the promise with an error message, while resolving with the correct result otherwise.

Project Structure
index.html → Main HTML file that loads the program.

style.css → Provides styling for the page.

script.js → Contains the JavaScript logic with the promise-based division function and test cases.

Features
Arrow function that returns a Promise.

Error handling for division by zero.

Dynamic output displayed on the webpage.

CSS styling for success (green) and error (red) messages.
